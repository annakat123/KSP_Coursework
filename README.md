# Курсовая работа по КСП

Небольшое приложение для оформления заявок на обслуживание машин и спецтехники.

## Идея проекта

Пользователь оставляет заявку, указывает данные машины, описывает проблему и при необходимости добавляет информацию по детали. Администратор смотрит заявки и меняет их статус: принято, не хватает данных или нужно переделать.

## Планируемые роли

- Клиент - создает заявки и смотрит их статус.
- Администратор - просматривает заявки и работает со справочником деталей.

## Что будет показано в курсовой

- Верстка страниц на HTML.
- Оформление через CSS.
- Простая динамика на JavaScript.
- Сохранение черновика заявки в LocalStorage.
- Обработка данных через PHP.
- REST API для заявок.
- Небольшая React-часть для работы с API.

## Структура

```text
api/        серверная часть и API
data/       данные для разработки
docs/       материалы для отчета
logs/       логи
public/     HTML, CSS и обычный JavaScript
src/        React-часть
uploads/    файлы пользователей
```

## Запуск

Пока проект можно открыть как обычные HTML-страницы. Главное  находится в `public/index.html`.
